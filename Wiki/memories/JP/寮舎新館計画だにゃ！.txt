== 寮舎新館計画だにゃ！ ==

<br>
母港・執務室<br>
朝早くから、大事な相談があると明石が執務室を訪ねてきた。<br>
<span style="color:#4eb24e;">明石：</span><br>
おはようにゃ～指揮官。お待たせしたにゃ！<br>
はい、これが新しい提案書——「寮舎新館建設計画」だにゃ～<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
——寮舎の…新館？明石が相談したい大事なことってこれか…<br>
<span style="color:#4eb24e;">明石：</span><br>
そうだにゃ！母港がどんどん発展してるから、もっと住み心地の良い環境を提供すべきだと思ったにゃ！<br>
ちょうどこの前、イングラハムが廃工場の爆破解体を行ったし、あそこなら広さも十分にゃ～<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
——ん？寮舎をリニューアルするのではなく、文字通り新館を作るのか？<br>
<span style="color:#4eb24e;">明石：</span><br>
もちろんにゃ～。新築にしたほうが儲けられるにゃ♪<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
——……<br>
（ごまかさずに素直に教えるのか…）<br>
<span style="color:#4eb24e;">明石：</span><br>
とにかく、新たな寮舎のコンセプトは、母港の仲間たちがよりよく休息をとって、変わらない日常を楽しんでもらうことにゃ<br>
戦闘も仕事も相当体力を消費するにゃ。もっと快適に過ごせる場所があったら、普段の調子もきっと良くなるはずにゃ！<br>
指揮官も、そう思わないかにゃ？<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
——それもそうだな。ゆっくり休めたら仕事も戦闘も捗る<br>
<span style="color:#4eb24e;">明石：</span><br>
だからみんなが満足できるよう、新館の部屋は入居者の好みに合わせて内装を仕上げるにゃ<br>
部屋の隅々まで、その子が好きそうなデザインにするって保証するにゃ！<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
——いわゆる「手ぶら入居」ってことか…<br>
<span style="color:#4eb24e;">明石：</span><br>
そうにゃ。指揮官、よくわかってるにゃ！<br>
もちろんみんなの寮舎だけど、指揮官がどうしてもというにゃら…<br>
中に指揮官専用スペースを作ることもできなくはないにゃ？<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
——それは今度考えておくよ…<br>
——とにかく提案は承認しよう。それで、いつから着工する？<br>
<span style="color:#4eb24e;">明石：</span><br>
すぐにでも始められるにゃ！あ。一つだけ事前に指揮官に伝えておきたいことがあるにゃ<br>
母港の現在の資材備蓄と技術力があれば、すぐに完成させるのも難しくにゃいけど…<br>
でも、すぐにみんなを入居させるのは無理にゃ<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
——どうして？<br>
<span style="color:#4eb24e;">明石：</span><br>
部屋の内装デザインには時間がかかるものにゃ。みんなモデルルームと同じ感じにするならすぐできるけど…<br>
明石、もう入居者の好みに合わせて内装を仕上げるって言っちゃったにゃ！<br>
だから順番待ちにして、一部屋ずつ入居してもらうしかないにゃ<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
——順番か……それも仕方ないか<br>
——ただそうなると一人目を誰にするかがかなり重要だ。なにせ新館の住み心地をフィードバックする役目を担うからな<br>
<span style="color:#4eb24e;">明石：</span><br>
それはもう考慮済みにゃ！メイドさんに手伝ってもらうことにしたにゃ～<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（ん？メイド？ベルファストのことか？ロイヤルメイドのメイド長ならたしかに完璧にこなせそうだが…）<br>
<br>
執務室のドアが開く音とともに、見慣れた姿が目に入ってきた。<br>
<span style="color:#4eb24e;">シリアス：</span><br>
ご機嫌麗しゅうございます。誇らしきご主人様<br>
シリアス、寮舎の最初の入居者となれて大変光栄に存じます<br>
<span style="color:#4eb24e;">明石：</span><br>
むふふふ、そういうことにゃ～。シリアスの好みに合わせて設計図を仕上げたから、あとは指揮官がサインするだけでいいにゃ！<br>

<br>
ちょっとだけ考えた後、書類にサインすることにした。<br>
<span style="color:#4eb24e;">シリアス：</span><br>
シリアスは必ずやこの務めを全ういたします。誇らしきご主人様！<br>
<br>
一ヶ月後、母港・執務室——<br>
<span style="color:#4eb24e;">明石：</span><br>
指揮官！寮舎新館の建設工事が完了したにゃ！それとシリアスの部屋の内装も終わったにゃ！<br>
もちろん、部屋の中は全てホルムアルデヒドなどの有害物質を含まない素材と家具を使ってるから、心配無用だにゃ<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
——あれ？シリアスは？<br>
<span style="color:#4eb24e;">明石：</span><br>
シリアスはもう部屋で指揮官を待ってるにゃ！さあ、早く会いに行くといいにゃ！<br>
