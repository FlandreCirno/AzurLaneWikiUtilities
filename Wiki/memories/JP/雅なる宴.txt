雅なる宴<br>
<br>
一　茶室にて<br>
<span style="color:#4eb24e;">海天：</span><br>
「揺れる松渓流で月を煎り、露含む梅のほとりで雲が立つ」<br>
ふぅ…香り高くていいお茶ですね<br>
<span style="color:#4eb24e;">ハウ：</span><br>
えっと…「東煌茶会」はここで合ってるかしら？<br>
あら、ドアが開いたままだわ<br>
<span style="color:#4eb24e;">海圻：</span><br>
東煌茶室へようこそ！一名様？<br>
<span style="color:#4eb24e;">ハウ：</span><br>
ええ<br>
<span style="color:#4eb24e;">海圻：</span><br>
ではこちらへどうぞ！<br>
<br>
海圻に案内され、ハウは古風な茶室へ入り、椅子に腰を下ろした。<br>
<span style="color:#4eb24e;">海天：</span><br>
いらっしゃいませ、ハウさん<br>
<span style="color:#4eb24e;">ハウ：</span><br>
海天さん、お邪魔するわ<br>
<span style="color:#4eb24e;">海天：</span><br>
来ていただき光栄です。海圻、お茶を持ってきてくれますか<br>
<span style="color:#4eb24e;">海圻：</span><br>
はいよ！ハウさん、どうぞー<br>
<span style="color:#4eb24e;">ハウ：</span><br>
ありがとう。私も手作りのクッキーを持ってきたからよかったらどうぞ<br>
<span style="color:#4eb24e;">海天：</span><br>
ハウさんはお客様なのに…恐縮です<br>
<span style="color:#4eb24e;">ハウ：</span><br>
大丈夫よ。仲間だもの<br>
<span style="color:#4eb24e;">海圻：</span><br>
海天姉ちゃん見て！このクッキー、「福」の文字の形をしてるよ<br>
そんでこっちは…灯籠にそっくり！ハウさん凄い！<br>
<span style="color:#4eb24e;">ハウ：</span><br>
東煌の仲間のところに訪れるからには、東煌の要素を取り入れてみようと思ってね。気に入ってもらえたなら嬉しいわ<br>
<span style="color:#4eb24e;">撫順：</span><br>
なになに？美味しいクッキー？私たちも食べたい！<br>
<br>
クッキーの香りに気づいたか、撫順はグロズヌイを連れて茶室へ入ってきた。<br>
<span style="color:#4eb24e;">撫順：</span><br>
わ！こんなに可愛いクッキーだと食べるのがもったいなく感じちゃうね…グロズヌイもそう思うでしょ？<br>
<span style="color:#4eb24e;">グロズヌイ：</span><br>
うん。でも食べたい<br>
<span style="color:#4eb24e;">ハウ：</span><br>
ふふ、遠慮しないで。今夜の宴会のために沢山作っておいたわ<br>
<span style="color:#4eb24e;">撫順：</span><br>
へへへ、グロズヌイが東煌のお茶を飲んでみたいって言うから連れてきたよ。別にこれはいたずらとかじゃないでしょ？<br>
<span style="color:#4eb24e;">海天：</span><br>
ええ、むしろ良いことをしてあげたと思いますよ。海圻、グロズヌイさんにもお茶をお願い<br>
<span style="color:#4eb24e;">グロズヌイ：</span><br>
ティーポットはこれか？ならグロズヌイ、自分で注げるよ<br>
<br>
そう言うと、グロズヌイはいつも使っている水筒にお茶を注いで、一気に飲み干した。<br>
<span style="color:#4eb24e;">海天：</span><br>
た、大変…！グロズヌイさん、火傷してませんか？<br>
<span style="color:#4eb24e;">グロズヌイ：</span><br>
…っぷ。この程度はへーきだよ<br>
<span style="color:#4eb24e;">撫順：</span><br>
ダメだよグロズヌイ！お茶はもっとゆっくり楽しむものだよ？飲む時も水筒ではなくこのコップで！<br>
<br>
撫順はグロスヌイから水筒を奪い取り、代わりに茶杯（チャハイ）を机に置いた。<br>
<span style="color:#4eb24e;">グロズヌイ：</span><br>
小さい。ごくごくできないからスッキリしない…<br>
<span style="color:#4eb24e;">撫順：</span><br>
これがいいの！鞍山姉さんから教わった飲み方を教えてあげるから、見てて<br>
<span style="color:#4eb24e;">海天：</span><br>
ふふふ、微笑ましいことですね。やはりお茶会を開いて正解でした<br>

<br>
二　庭園にて<br>
東煌風庭園セット・某所<br>
<span style="color:#4eb24e;">鎮海：</span><br>
千歳さん、目線をこちらに――はい、そのまま（パシャ）<br>
うーん、きれいに撮れましたけど、あともう一つ、「これだ」といったものがあればいいんですが……<br>
<span style="color:#4eb24e;">千歳：</span><br>
いい写真を撮るの…難しい…ひ、ひゃう！<br>
<br>
立ち上がろうとしたところ、千歳の髪が木の枝に引っかかってしまった。<br>
<span style="color:#4eb24e;">鎮海：</span><br>
あらら…千歳さん、動かないでください。ここは私に任せて<br>
<span style="color:#4eb24e;">千歳：</span><br>
…うぅ、ごめん…お願い…<br>
<br>
冷静沈着な鎮海の対処で、千歳の髪は枝から離れ、事なきを得た。<br>
<span style="color:#4eb24e;">鎮海：</span><br>
これで大丈夫ですわ。こんなきれいな髪ですもの、何もなくて本当によかったです<br>
<span style="color:#4eb24e;">千歳：</span><br>
ありがとう…髪の手入れは、全部千代田がしてくれているよ<br>
今回の記念写真も――<br>
「せっかく髪も整えて綺麗な服で着飾ったのに、記念に写真に残さないのはもったいないよ」って千代田が…<br>
<span style="color:#4eb24e;">鎮海：</span><br>
姉妹仲の睦まじいことです。今度は千代田さんも誘って一緒に写真撮ってはいかがでしょう？<br>
<span style="color:#4eb24e;">千歳：</span><br>
いいと思う…<br>
<span style="color:#4eb24e;">鎮海：</span><br>
あら、今ので閃きましたわ。千歳さん、もう少しだけ付き合っていただけますか？<br>
<span style="color:#4eb24e;">千歳：</span><br>
いいけど…私はどうすれば…？<br>
<span style="color:#4eb24e;">鎮海：</span><br>
髪の毛をそちらの枝に軽くかけてもらってもいいですか？<br>
<span style="color:#4eb24e;">千歳：</span><br>
えっ？さ、さっき引っかかったばかりなのに…？<br>
<span style="color:#4eb24e;">鎮海：</span><br>
はい、ですので引っかからないように「軽く」かける、ということですわ<br>
<span style="color:#4eb24e;">千歳：</span><br>
わ、わかった…ええと、こ、これでいい…？<br>
<br>
（パシャ）<br>
<span style="color:#4eb24e;">鎮海：</span><br>
ふふ、良い感じに決まりましたわ。この一枚の名前は「梢上佳景」にしようかしら<br>
<br>
千歳と別れた後、何やら騒々しい音を聞きつけた鎮海は、そちらの様子を見に向かうことにした。<br>
<span style="color:#4eb24e;">応瑞：</span><br>
力を込めすぎないで、ゆっくり、軽く、ですよ<br>
<span style="color:#4eb24e;">ボイシ：</span><br>
わかった…ゆっくり…あ、あぅ<br>
破れちゃった…うぅ、やっぱ無理だよ…<br>
<span style="color:#4eb24e;">応瑞：</span><br>
焦らないでください。ボイシさんはただ慣れてないだけですから<br>
ほら、紙は破れちゃいましたけど、上手に書けていますよ？次はもう少し力を抜けばきっとうまくいきますよ<br>
<span style="color:#4eb24e;">ボイシ：</span><br>
う、うん！もう一回、やってみる…<br>
<br>
画仙紙を広げ、ボイシは真剣な面持ちで筆を取り…<br>
<span style="color:#4eb24e;">応瑞：</span><br>
楽にしてください。ボイシさんならきっとうまくやれますよ<br>
<span style="color:#4eb24e;">ボイシ：</span><br>
ふーはー…深呼吸…ふーはー…うん…！<br>
…できた！<br>
<span style="color:#4eb24e;">応瑞：</span><br>
うんうん。よくできましたよ。言った通りでしょ？<br>
<br>
（パシャ）<br>
<span style="color:#4eb24e;">鎮海：</span><br>
ふふ、ではこの一枚は「苦尽甘来」としましょうか<br>

<br>
三　楽室にて<br>
音楽室で、ローンは一生懸命ピアノを弾いていた。<br>
せせらぎのように美しい音色が、逸仙とティルピッツを優しく包んだ。<br>
そして一曲弾き終わると――<br>
<span style="color:#4eb24e;">ローン：</span><br>
今回はどうでしたか？<br>
<span style="color:#4eb24e;">逸仙：</span><br>
ローンさん、前回と比べると見間違えるほどの成長っぷりですね。<br>
<span style="color:#4eb24e;">ティルピッツ：</span><br>
ええ。これなら間違いなく宴会でも注目の的ね<br>
<span style="color:#4eb24e;">ローン：</span><br>
よかったです～。これで指揮官のための独奏会を開くという目標にまた一歩近づけました～<br>
とはいえ、まだまだ曲の引き出しが足りていませんね<br>
<span style="color:#4eb24e;">逸仙：</span><br>
日々練習の積み重ねが大事です。焦らず、少しずつ身につけていきましょう<br>
<span style="color:#4eb24e;">ローン：</span><br>
もちろん。今は今夜の宴会に集中しませんと<br>
指揮官にも捧げる演奏ですから、ミスは絶対に許されませんね～<br>
ではもう一回練習しましょうか。逸仙さん、よろしくお願いします<br>
<span style="color:#4eb24e;">逸仙：</span><br>
喜んで。ローンさんが段々上手くなっていっているので私も嬉しいです<br>
<span style="color:#4eb24e;">ティルピッツ：</span><br>
ローン、今回はやはり合奏しよう<br>
<span style="color:#4eb24e;">ローン：</span><br>
ふふ、やっと決心がつきました？<br>
<span style="color:#4eb24e;">ティルピッツ：</span><br>
指揮官の前で披露してあげたい気持ちに嘘はつけないわ<br>
それに、ローンの演奏を聞いて私もじっとしてはいられなくなった<br>
そろそろ一歩踏み出す時かもしれないわね<br>
<span style="color:#4eb24e;">ローン：</span><br>
決心がついたからには、一緒に演奏しましょうか<br>
言っておきますけど、ティルピッツさんに合わせてパフォーマンスを落とすつもりはありませんよ<br>
<span style="color:#4eb24e;">ティルピッツ：</span><br>
言ってくれる。こちらもちゃんと練習してきたわ<br>
それに、本番まで逸仙には何回か聞いてもらえるから<br>
逸仙、私にもアドバイスをお願いできるかしら？<br>
<span style="color:#4eb24e;">逸仙：</span><br>
もちろん喜んで。お二人の合奏で今夜の宴会はより素晴らしいものになるでしょう<br>
<span style="color:#4eb24e;">ローン：</span><br>
では、早速練習を始めましょうか<br>
<br>
旋律が鳴り響く――ローンのピアノとディルピッツのフルートが織りなす合奏は美しい音を奏で続けていた。<br>

四　宴会にて<br>
母港・宴会ホール<br>
<span style="color:#4eb24e;">ワシントン：</span><br>
いよいよか…<br>
（散々練習してきたんだ。今なら自然な笑顔になれるはず…）<br>
（落ち着けワシントン、みんながしているように、宴会を楽しめばいいんだ！）<br>
<span style="color:#4eb24e;">ル・マルス：</span><br>
ご、ごきげんよう！<br>
<span style="color:#4eb24e;">ワシントン：</span><br>
（こいつ、アタシと同じで緊張している！？…確かヴィシアのル・マルスだっけ…）<br>
<span style="color:#4eb24e;">ル・マルス：</span><br>
あ、あの…ユニオンのワシントンさん、こ、こんばんは！<br>
<span style="color:#4eb24e;">ワシントン：</span><br>
あんた、すげー緊張してるだろ？<br>
<span style="color:#4eb24e;">ル・マルス：</span><br>
えぇ！？な、なんで分かったんですか！？<br>
<span style="color:#4eb24e;">ワシントン：</span><br>
そりゃ一目で分かるだろ<br>
<span style="color:#4eb24e;">ル・マルス：</span><br>
も、申し訳ないです。お見苦しいところを…<br>
<span style="color:#4eb24e;">ワシントン：</span><br>
（ふぅ…なんかアタシよりも緊張してるやつを見たら、だいぶ落ち着いてきたな…）<br>
こういうところは初めてか？<br>
<span style="color:#4eb24e;">ル・マルス：</span><br>
はい！護衛として入口で当直することは多かったんですけど、客として参加するのは…<br>
しかもドレスを着たり、他にも気をつけないところがたくさんあって…もういっぱいいっぱいです…うう…<br>
<span style="color:#4eb24e;">ワシントン：</span><br>
心配する必要はないさ。ここに来てるのはみんな仲間だ、そこまで気にしないでもいいだろ<br>
逆にそこまで畏まられたら、ほかの子も話しかけづらいと思うぞ<br>
<span style="color:#4eb24e;">ル・マルス：</span><br>
ああ、言われてみればそうですね！<br>
<span style="color:#4eb24e;">ワシントン：</span><br>
だろ？アタシのようにスマイルでも振りまいてみたら気も楽になると思うぞ<br>
<span style="color:#4eb24e;">ル・マルス：</span><br>
…はい！大分楽になりました！<br>
ワシントンさん、どうもありがとうございました！段々慣れてきました！<br>
<span style="color:#4eb24e;">ワシントン：</span><br>
そりゃよかった。じゃそのまま宴会を楽しむことだな。アタシもアタシで宴会の料理を楽しむつもりだ<br>
（さっきアタシの「笑顔」は特に何も言われなかった…つまり、「自然にできた」ってこと？）<br>
（やった！ってちょっと待った…さっきはどうやったんだっけ…）<br>
（あああ、早く思い出せよアタシ！）<br>
<br>
他人のピンチを助けたワシントンだが、どうやら今度は自分がピンチのようだ…<br>
そして宴会が始まり――<br>
<span style="color:#4eb24e;">ハウ：</span><br>
出来立てのクッキーよ。みんな、遠慮しないで召し上がって<br>
<span style="color:#4eb24e;">グロズヌイ：</span><br>
あの灯籠の形してるのがいい…はむ<br>
<span style="color:#4eb24e;">千歳：</span><br>
この扇子の形の…いい感じ。これにする…<br>
<span style="color:#4eb24e;">ボイシ：</span><br>
ありがとう…じゃボイシはこの文字の形をしたのにする…<br>
<span style="color:#4eb24e;">ティルピッツ：</span><br>
ふぅ…ステージに上がって皆の前での演奏、か<br>
<span style="color:#4eb24e;">ローン：</span><br>
まだ緊張してますか？<br>
<span style="color:#4eb24e;">ティルピッツ：</span><br>
平気よ。…多分緊張より、高揚といったところだわ<br>
<span style="color:#4eb24e;">ローン：</span><br>
ふふ、ならちょうどいいですね<br>
その感情を込めて演奏して、旋律に乗せて思いっきり解き放ちましょう――<br>
<br>
（パシャ）<br>
<span style="color:#4eb24e;">鎮海：</span><br>
「良きかな賓客雲集の席、語りきれぬ愉快な宴。響き渡る管弦合奏の曲、聞こえるは旋律の妙」<br>
この一枚は、「雅なる宴」と名付けましょう<br>
宴会が始まったことですし、私の記念写真の仕事もこの辺にして楽しむとしましょう<br>
ということで指揮官、私と一曲、いかがかしら？<br>
