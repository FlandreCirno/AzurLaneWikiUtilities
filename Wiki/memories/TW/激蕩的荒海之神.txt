激蕩的荒海之神<br>
<br>
1 文件大雪崩<br>
指揮室內，埃吉爾正在和我一起處理今天工作所需使用的文件。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
已經整理完的我放在這裡了。<br>
不過……我說啊，今天的工作是不是有點太多了？<br>
<br>
埃吉爾一邊說著，一邊皺著眉看向桌面上的文件。<br>
順著她的視線看去，演習報告、會議資料、倉庫清單……等等，已經堆到了一人多高。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
這也是沒辦法的事情，畢竟今天是星期一……<br>
不過，居然一口氣就到了這麼恐怖的量，也有點出乎我的預料了。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
真是悲慘……沒辦法，就讓我來幫你吧，指揮官。<br>
記得要好好感激我，呵呵呵呵呵~<br>
<br>
埃吉爾抬手將長髮別在耳後，隨後優雅地坐回到辦公桌前，開始了她的工作。<br>
雖然剛開始有些小插曲，但埃吉爾很快便上手了秘書艦的工作。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（就像今天，如果沒有她在的話，這些文件想必可以讓我工作到天明吧……）<br>
<br>
無言地望了望那一疊混雜著無數硬殼合訂本的文件山。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（嗯……還得再快一點。）<br>
<br>
許久之後，直到夜色將窗外籠罩，桌上的文件數量依然還有許多。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（這個工作量不加班是不可能了，不過對我來講也是常事。）<br>
（但是埃吉爾那邊……）<br>
<br>
埃吉爾並未察覺到我的視線，她伸了個懶腰後打了個哈欠。<br>
一整天的工作似乎讓她很疲倦。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
埃吉爾，辛苦了。妳先回去休息吧，接下來交給我自己處理就好。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
指揮官，你是在跟我開玩笑嗎？<br>
<br>
眼前的少女挑了挑眉，用銳利的視線看向我，讓人感受到那屬於深海之神的威壓。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
現在的情況我怎麼可能安心離開？我去幫你泡一杯咖啡，我們繼續。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（果然拗不過她嗎……）<br>
<br>
無奈地對她點點頭，算是同意了她的「加班申請」。<br>
少女隨即便從座椅上起身，準備去泡咖啡。然而，就在這時出現了意外——<br>
不知道是因為埃吉爾起身時低著頭，導致角的邊緣撞到了文件的緣故，還是因為別的什麼……<br>
那堆夾雜著硬殼文件夾的文件堆開始了它的崩塌。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（不好，這樣會砸到埃吉爾！）<br>
<br>
身體比意識先一步動作，將埃吉爾拉入了懷中。<br>
大量的文件崩落，雖然這意味著等下又要重新整理一遍，但……<br>
還好，這些東西沒有砸到她。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
埃吉爾，沒事吧？<br>
<br>
雖然在文件崩塌之前就已經把埃吉爾拉過來了，但還是下意識追問了一句。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
欸……？<br>
<br>
懷中的少女一臉迷茫，似乎完全沒反應過來發生了什麼事。<br>
此時此刻，她因為被拉住時的慣性作用，整個人半躺在我的懷抱中。<br>
能夠清楚地看到她急促呼吸時胸前的起伏，以及正在飛快染上紅暈的皮膚。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
指、指揮官，你、你你你……！<br>
<br>
在短暫的呆愣過後，她立刻從懷抱中掙脫了出來，飛也似地去到了沙發的另一邊。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（這是在害羞……嗎？不對，現在不是感嘆這個的時候。）<br>
<br>
我低頭看向地上散落的文件，拿起一份擺回了桌子上。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
………………<br>
<br>
在害羞的情緒消褪後，埃吉爾沉默地看著我撿起文件的動作。<br>
很遺憾，原本勉強排列好的文件因為崩塌的緣故全部被打亂，根本不知道一開始歸屬於哪個文檔。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
沒關係，埃吉爾沒事就行……當務之急是把文件整理起來。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
我來吧。這種程度而已，我一個人就能處理好。<br>
反而是指揮官你現在看起來可是疲憊不堪~？不過我理解，工作太多也是沒辦法的事。<br>
<br>
少女故作輕鬆地「嘲笑」著我。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
所以，你還是去休息比較好。再說了，這本來就是我的責任。<br>
<br>
埃吉爾說完後，擺出了不容拒絕的態度，把我趕到了沙發上。<br>
她則開始努力收拾那些散落的文件。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（埃吉爾……這是在自責吧？）<br>

2 再整理<br>
雖然被埃吉爾趕到了沙發上休息，但不論如何……我都不能眼看著她一個人忙碌。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（至少要堅持到把所有散落的文件全部搬到桌子——）<br>
<br>
五分鐘後——<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（不太樂觀啊……光是把文件重新搬上桌就花了五分鐘。）<br>
（這如果埃吉爾堅持一個人整理的話……得想個辦法。）<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
好了，接下來就交給我吧。<br>
將這些文件重新整理好……倒也算是個不小的挑戰，那麼，就見識一下埃吉爾的力量吧！<br>
<br>
就在我思考如何才能不著痕跡的幫上埃吉爾的時候，指揮室外突然響起了敲門聲。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（對了……昨天彼得·史特拉塞說今晚會送來演習報告。）<br>
（她應該可以幫我一把。）<br>
<br>
想到這裡，我坐直了身體，隨後讓史特拉塞進入了指揮室。<br>
趁她將演習報告遞給我的時候，我用眼神瞄向了一旁。<br>
雖然一開始有些困惑，不過在看見埋頭整理文件的埃吉爾後，她便露出了然的表情。<br>
<span style="color:#4eb24e;">彼得·史特拉塞：</span><br>
……以上就是報告的內容，指揮官，如果確定沒有問題，就在這裡簽字吧。<br>
至於其他的……埃吉爾，我有更效率的方法可以分類文件，要試試看嗎？<br>
<br>
史特拉塞話題一轉，將話題引向了埃吉爾。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
好吧，彼得，快點告訴我怎麼做！<br>
不過，我不會讓妳插手整理的。<br>
<span style="color:#4eb24e;">彼得·史特拉塞：</span><br>
那當然，畢竟這是妳的工作。時間寶貴，讓我們開始吧？<br>
<br>
經過史特拉塞的旁側敲擊，在一個小時的「奮戰」後，文件整理的工作已經進入了尾聲。<br>
當埃吉爾還在聚精會神處理剩下的文件時，彼得·史特拉塞藉口有私事要請教我，將我叫離了指揮室。<br>
<span style="color:#4eb24e;">彼得·史特拉塞：</span><br>
呼呼，指揮官，剛剛在指揮室裡我就看你好像有話想問我呢。<br>
所以我就主動把你叫出來了……有什麼想問的就快問吧？我預留給報告的時間不多了哦。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
不愧是善解人意的史特拉塞小姐，其實我想問一些關於埃吉爾的事情——<br>
<span style="color:#4eb24e;">彼得·史特拉塞：</span><br>
啊~原來如此，是想跟埃吉爾打好關係嗎？可是，我看指揮官明明和埃吉爾關係很不錯。<br>
……嗯，這樣的話，指揮官，您可以試試看一些精緻的小東西。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
精緻的小東西……首飾一類的嗎？<br>
<span style="color:#4eb24e;">彼得·史特拉塞：</span><br>
沒錯，相信您一定可以憑直覺找到適合的禮物。<br>
不過……居然是問禮物嗎……呵呵~<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
嗯？<br>
<span style="color:#4eb24e;">彼得·史特拉塞：</span><br>
沒什麼。那麼祝你好運了，指揮官~<br>
<br>
說完這些，彼得·史特拉塞便離開了。<br>
<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
（嗯……我也該回去陪埃吉爾工作了。）<br>

我看向另一旁的辦公桌。<br>
此刻，埃吉爾正用手肘撐著自己頭，一臉倦色。<br>

<br>

<span style="color:#3498DB;" class="shikikanname">指挥官</span>：<br>
從一開始就是認真的。<br>
<span style="color:#4eb24e;">埃吉爾：</span><br>
這樣太狡猾了！指揮官，你真是……！<br>
<br>
之後，和埃吉爾約好了休息日一起出門後，結束了今天的工作。<br>
