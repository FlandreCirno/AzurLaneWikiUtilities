有些故事沒有開端，有些故事沒有結尾。而有些故事，就這樣發生了————<br>
■■■■年■月■日 海域坐標■■■■ ■■■■<br>
轟————————————！<br>
<span style="color:#4eb24e;">小克里夫蘭：</span><br>
敵襲出現了？！海倫娜，妳沒事吧！<br>
<span style="color:#4eb24e;">小海倫娜：</span><br>
我沒事……但是貨物又被搶走了！<br>
<span style="color:#4eb24e;">小克里夫蘭：</span><br>
怎麼會這樣………已經是本週第三次襲擊了。<br>
再這樣下去，我們的甜點供應就要徹底中斷了啊！<br>
<span style="color:#4eb24e;">小聖地牙哥：</span><br>
主動出擊啊，主動出擊！事到如今只能主動出擊了！<br>
我們一定要把搶奪貨船的人抓出來！<br>
<span style="color:#4eb24e;">小海倫娜：</span><br>
總之還是先收集情報吧……沒有情報的話是抓不到犯人的哦。<br>
<span style="color:#4eb24e;">小克里夫蘭：</span><br>
那就這麼決定了，我帶領更多的量產型加強護航力度。<br>
海倫娜去跟其他陣營交換情報。甜點供應中斷的話，其他陣營應該也很困擾才是。<br>
<span style="color:#4eb24e;">小聖地牙哥：</span><br>
那我呢，那我呢？！<br>
<span style="color:#4eb24e;">小克里夫蘭：</span><br>
聖地牙哥就負責去寫給薩拉托加的報告好了！也許她那邊會知道什麼事！<br>
<span style="color:#4eb24e;">小聖地牙哥：</span><br>
了解啦~！給小加加寫報告的事就交給我吧！<br>
<span style="color:#ec5d53;">小施佩：</span><br>
呼呼呼……甜點，美味的甜點……<br>
再多收集一些吧~<br>

幾日後——————<br>
<span style="color:#4eb24e;">小企業：</span><br>
海倫娜，我帶領白鷹的支援抵達了哦。<br>
<span style="color:#4eb24e;">小柴郡：</span><br>
快看快看，皇家的支援也抵達了喲————<br>
<span style="color:#4eb24e;">小聲望：</span><br>
您好，海倫娜小姐，我帶來了皇家的支援與陛下的問候。<br>
<span style="color:#4eb24e;">小光輝：</span><br>
為了愛與和平，我們一定要盡快平息此次甜點危機才行！<br>
<span style="color:#4eb24e;">{namecode:170}：</span><br>
重櫻艦隊也來了哦，真沒想到會發生這種事。<br>
<span style="color:#4eb24e;">{namecode:169}：</span><br>
甜點供應減少，連宴會都要開不起來了……<br>
<span style="color:#4eb24e;">小天城：</span><br>
好吃的團子也沒有了……必須要盡快，阻止事態的惡化！<br>
<span style="color:#4eb24e;">小加加？：</span><br>
滴滴滴——這裡是小加加來自後方的最新情報！<br>
是鐵血！鐵血就是這一系列甜點運輸船襲擊事件的幕後黑手！<br>
我在去海邊遊樂園玩的時候，偶然目擊到了歐根親王進行甜點交易的現場！<br>
當時我只顧偷看交易，卻忽略了從背後而來的另一個同夥。<br>
……等我醒過來之後，交易已經結束了！<br>
<span style="color:#4eb24e;">小海倫娜：</span><br>
不愧是薩拉托加，真是很有意義的情報……<br>
襲擊團伙裡還有其他人，不過另一個人究竟是……<br>
<span style="color:#4eb24e;">小克里夫蘭：</span><br>
緊急情況！運送甜點的運輸船又被襲擊了！<br>
量產型艦隊已經與敵人交戰了，我正在前往現場，大家也一起支援吧！<br>
這次一定要抓到幕後黑手！<br>
{{折叠面板|内容结束}}

<span style="color:#4eb24e;">小克里夫蘭：</span><br>
抓住妳了！搶奪甜點的犯人！<br>
<span style="color:#4eb24e;">{namecode:170}：</span><br>
哼哼哼~還是當場抓獲，剩下的事就去監獄裡交代吧。<br>
<span style="color:#4eb24e;">小聲望：</span><br>
立刻放下武器把甜點交出來，否則……我們就要被迫採取武力措施了。<br>
<span style="color:#ec5d53;">小施佩：</span><br>
我是不會把甜點交出來的喲。<br>
鐵血也不止是啤酒和香腸，明明甜點也很重要……但是很少有人注意到這一點。<br>
……所以我制定了一個計劃，一個可以把大海變成甜點的偉大計劃！<br>
現在已經進行到最後一步了……<br>
很快，大海就要被甜點覆蓋了……不要阻止我！<br>
<span style="color:#4eb24e;">小企業：</span><br>
……雖然沒太聽懂，但是我感覺到了危險。<br>
<span style="color:#4eb24e;">小聖地牙哥：</span><br>
甜點是很重要，但是大海也很重要哦！<br>
<span style="color:#4eb24e;">小天城：</span><br>
……蔚藍的大海讓人心曠神怡，而且也是海鮮賴以生存的家園……<br>
不會讓妳肆意妄為的……！<br>
<span style="color:#ec5d53;">小施佩：</span><br>
那就放馬過來吧。我可是很強的，被打疼了也不許哭哦~<br>
<br>
<span style="color:#4eb24e;">小克里夫蘭：</span><br>
大家開火的時候注意，不要打到甜點運輸船了！<br>
<span style="color:#4eb24e;">小海倫娜：</span><br>
……那些裂縫好像不是我們打的……是小施佩自己在破壞那些運輸船！<br>
<span style="color:#4eb24e;">小聖地牙哥：</span><br>
啊啊啊啊，運輸船都裂開了，甜點都流進海裡了！<br>
<span style="color:#4eb24e;">小光輝：</span><br>
……然后海水本身正在變成甜點？！<br>
<span style="color:#4eb24e;">小天城：</span><br>
…………好甜，好好喝……（咕嚕）<br>
<span style="color:#4eb24e;">{namecode:170}：</span><br>
嗚啊啊天城姐姐，這個不能喝啊！<br>
<span style="color:#4eb24e;">小企業：</span><br>
這究竟是……你究竟做了什麼？！<br>
<span style="color:#ec5d53;">小施佩：</span><br>
將大海，變為甜點之海，我原本是這麼計劃的……<br>
可惜不能如願了…………<br>
這樣的話，我就要在此自沉，沉入無窮無盡的甜點之海裡！<br>
<span style="color:#4eb24e;">小克里夫蘭：</span><br>
……等！<br>
<span style="color:#ec5d53;">小施佩：</span><br>
再見了，我不會再回來了——<br>
<br>
就這樣，甜點危機得到了解決。<br>
在小施佩自沉的地方，到現在也會湧出源源不斷的甜點，成為了廣受好評的旅遊勝地。<br>
至於小施佩，有人說她已經前往甜點的世界了，有人說她被小歐根救了回去。<br>
不論如何，沒有人再見過她的身影。<br>
總而言之，有些故事沒有開端，有些故事沒有結尾。而有些故事，就是這樣結束了————<br>
——THE END——<br>
