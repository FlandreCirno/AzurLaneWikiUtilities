五 對手戲<br>
演習海域·廢棄設施<br>
庫珀與英格拉罕在黑暗的設施中前進著。<br>
<span style="color:#4eb24e;">庫珀：</span><br>
照明設施啟動！<br>
哇……這裡的陷阱佈置的還真像那麼回事啊。<br>
<span style="color:#4eb24e;">英格拉罕：</span><br>
是有點機關屋的感覺，不過缺少點機械感的要素，跟廢棄設施的主題不是很貼合……<br>
不對……現在不是研究這些的時候。<br>
發現布里斯托爾了嗎？<br>
<span style="color:#4eb24e;">庫珀：</span><br>
順著黑暗中搜索……仔細搜索…………啊！布里斯托爾發現！<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
（驚！）<br>
<span style="color:#4eb24e;">庫珀：</span><br>
布里斯托爾！不要跑！<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
冒牌貨快離開，冒牌貨快離開！這裡沒有妳要找的人——<br>
<span style="color:#4eb24e;">庫珀：</span><br>
到底發生什麼事了，怎麼我回來之後到處都怪怪的！<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
欸？妳也是剛從外面執行任務回來嗎！<br>
<span style="color:#4eb24e;">庫珀：</span><br>
是啊！<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
太好了……我還以為就剩我一個人了呢！<br>
妳去過港區那邊了嗎？<br>
<span style="color:#4eb24e;">庫珀：</span><br>
還沒有……到底發生什麼事了？只剩妳一個人指的是什麼？<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
嗚嗚嗚……港區遭遇了敵人襲擊！因為事發突然，大家都沒來得及撤離……<br>
指揮官在襲擊中受傷，我也失去了跟港區的通訊聯絡。<br>
<span style="color:#4eb24e;">庫珀：</span><br>
啊…………這還真是，太糟糕了。<br>
（英格拉罕那傢伙怎麼不把話說明白啊，我就知道事情是這樣子！）<br>
那妳現在在廢棄的設施裡做什麼呢？<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
我本來收到了緊急任務來這裡集結兵力準備反擊……沒想到這裡也淪陷了！<br>
不久之前我還遇到了塞壬棋子在搜捕我。<br>
還好我設下了機關，才拖延到現在……<br>
不論如何，我們都要在這裡堅守下去，這是我收到的最後任務！<br>
<span style="color:#4eb24e;">庫珀：</span><br>
原……原來如此？<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
堅持守住，就有辦法！我這不是等來了支援的妳嗎！<br>
只要繼續守下去，趕過來的同伴一定會越來越多，我們就能救援被困港區的同伴們了！<br>
<span style="color:#4eb24e;">英格拉罕：</span><br>
欸？堅定守住廢棄基地是什麼設定？<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
英格拉罕！通訊恢復了啊，妳那邊怎麼樣，還安全嗎？<br>
<span style="color:#4eb24e;">英格拉罕：</span><br>
啊……嗯，我還算安全，通訊也沒問題，妳放心說吧。<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
之前用暗號跟妳通訊的時候，妳不是用暗號回復“在廢棄基地附近躲雨”嗎。<br>
下雨的意思是敵人襲擊，送傘的意思是請求支援。<br>
妳現在無法送傘，但是先讓我躲雨，不就是先在這裡堅守，等待後續支援的意思嗎！<br>
<span style="color:#4eb24e;">英格拉罕：</span><br>
啊……………………<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
不愧是英格拉罕！多虧妳之前在突發聯絡時用的暗語，我才能這麼快理解事態的嚴重性並且做出正確的應對！<br>
<span style="color:#4eb24e;">庫珀：</span><br>
嗯……不愧是英格拉罕，讓簡單的事情完美複雜化了。<br>
<span style="color:#4eb24e;">英格拉罕：</span><br>
我再去給報告補上幾頁，先掛了……！<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
咦……？報告？簡單的事情複雜化是指什麼？<br>
<span style="color:#4eb24e;">庫珀：</span><br>
其實————<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
我有點暈……<br>
<span style="color:#4eb24e;">庫珀：</span><br>
沒事，我理解。<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
之前遇到的姐姐……也是真的姐姐嗎？<br>
<span style="color:#4eb24e;">庫珀：</span><br>
是這樣的哦。<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
之前踩上陷阱的人呢……？<br>
<span style="color:#4eb24e;">庫珀：</span><br>
好像是因為演習暫停從前線返回的同伴們。<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
嗚嗚嗚，感覺好丟人……給大家添麻煩了。<br>
<span style="color:#4eb24e;">庫珀：</span><br>
沒事沒事，大家都不會生氣的啦。<br>
倒是英格拉罕那傢伙真的應該好好檢討一下才是。<br>
走吧，我們該回去了。記得之後也給指揮官寫一份事件報告書哦。<br>
<span style="color:#4eb24e;">布里斯托爾：</span><br>
事件……報告書？！是，我明白了！<br>
<br>
翌日，在指揮室的辦公桌上出現了一份名為《演習神秘事件調查》的報告書。<br>
「……對艦船而言，最深刻的恐懼就是對未知的恐懼……」<br>
「……在那廢棄的基地中，沉睡的敵人等待著夢境降臨……」<br>
「……群星已到達特定的位置，真相終有一天會降臨港區……」<br>
附帶：《英格拉罕的檢討書》，艾倫·M·桑拿、庫珀監製。<br>
